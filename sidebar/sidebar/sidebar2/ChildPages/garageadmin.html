Virtual Garage Admin Interface

Commands for virtual garage console:

contents : prints the current contents of the garage
check_in owner reg_no make time_in_garage number_of_faults : using the command check_in and providing the 5 parameters will create a car with these attributes
fix reg_no : use the fix command and give the registration number of the car to fix
check_out reg_no : remove the car with the given registration number from the garage
bill reg_no : calculate and print the bill for the given registration number
